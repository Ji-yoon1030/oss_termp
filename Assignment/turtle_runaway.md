# Turtle Runaway Game Explanation

### 득점 방법
파란 거북이가 빨간 거북이를 잡으면 득점!(1점)
파란 거북이가 빨간 거북이를 잡는 순간 빨간 거북이는 멀리로 도망간다!

### 게임 오버
거북이들이 화면 밖으로 나가면 GAME OVER!
